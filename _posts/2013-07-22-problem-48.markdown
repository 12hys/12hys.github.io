Solution to Problem 48. Written in Python.

Problem: The series, 1^1 + 2^2 + 3^3 + … + 10^10 = 10405071317. Find the last ten digits of the series, 1^1 + 2^2 + 3^3 + … + 1000^1000.

```python
#!/usr/local/bin/pypy

sum_num = 0
for x in range(1, 1001):
    sum_num += pow(x, x)

print str(sum_num)[-10:]
```
